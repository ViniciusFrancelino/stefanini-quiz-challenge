# 🚀 Stefanini Quiz Challenge

## 📌 Sobre o Projeto

O **Stefanini Quiz Challenge** é uma aplicação web interativa desenvolvida para apresentação acadêmica, com o objetivo de demonstrar conhecimentos em desenvolvimento full stack.

A plataforma permite que usuários respondam a um quiz dinâmico, com armazenamento das respostas em banco de dados MySQL e exibição de um ranking em tempo real dos participantes com melhor desempenho.

O projeto foi desenvolvido para simular um desafio corporativo, destacando conceitos técnicos e boas práticas de desenvolvimento.

---

## 🎯 Objetivos

- Demonstrar habilidades em desenvolvimento web
- Implementar integração com banco de dados MySQL
- Aplicar conceitos de persistência de dados
- Desenvolver lógica de pontuação automatizada
- Exibir leaderboard com ranking dos participantes
- Praticar trabalho em equipe utilizando versionamento com Git

---

## 🛠️ Conceitos Aplicados

- Desenvolvimento Full Stack
- Integração Frontend + Backend
- Modelagem de Banco de Dados
- Manipulação de Dados com SQL
- Controle de Versão com Git
